# Getting started with Web Development

**what is web development and why ?**

to make a website

**what is website ?**

collection of webpages

**what is webpage ?**

it is an electronic document which contains text,hyperlinks and media.

**how to make a webpage ?**
to make a webpage we need some technologies like

- HTML -> structure
- CSS -> styling
- JavaScript -> Dynamicness or logic

- Jquery(optional) -> library of JS
- Bootstrap(optional) -> framework of CSS

## HyperText Markup Language

hypertext -> text + link
markup langauge -> build with tag eg. XML,HTML

**What are Tags ?**
building block of a website !

there are two types of tags :
1. closing tag eg. head,body,mark etc. (99%)
2. self-closing tag eg. hr,br,meta,img,input etc. (1%)

### closing tag
**syntax** : `<tagname attribute="value">content</tagname>`

- **Note: attributes are optional**

### self-closing tag
**syntax** : `<tagname>`

## Basic Structure of HTML Document

```html
<html>
    <head>
        <title>My Website</title>
    </head>
    <body>
        Any content
    </body>
</html>
```

## Modern Structure (HTML5)

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>MyTitle</title>
</head>
<body>
    Any Content
</body>
</html>
```
